# Cheat Ninja Legends

SWF file collection for Ninja Legends cheats (Tested on PC Only).

## Getting Started

### Installing

* Download `NinjaLegends.swf` file
* Paste the downloaded swf file on `C:\Program Files (x86)\Ninja Legends` folder
* Enjoy the game

### Cheat List

* One hit kill for hunting house
